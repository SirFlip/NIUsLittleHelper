Sie benutzen nun die aktuellste Version von "NIU's little helper" ... grandios!

Diese Features wurden seit der letzten Aktualisierung von uns implementiert:

  • Alle Dienste, Ambulanzen und Kurse auf der Startseite können jetzt in den eigenen Kalendar (Google, Yahoo, Outlook) exportiert werden.
  • Kurse können jetzt auch auf der Kursseite in den eigenen Kalendar exportiert werden.
  • Funktionäre: E-Mail Button direkt in das "Memo LAST" und Erinnerungs-Memo Interface integriert.
Den kompletten Versionsverlauf können Sie auf unserer Github Projektseite einsehen.

Wir wünschen Ihnen viel Spaß mit der neusten Version unserer Erweiterung!
