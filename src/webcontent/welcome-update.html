Sie benutzen nun die aktuellste Version von "NIU's little helper" ... grandios!

Diese Features wurden seit der letzten Aktualisierung von uns implementiert:

  • Alle Dienste, Ambulanzen und Kurse können jetzt in den eigenen Kalendar (Google, iCal, Outlook) exportiert werden.

  • FunktionärInnen: Bei Änderungen im Dienstplan durch Dienstführende gibt es nun die Möglichkeit Autocomplete-Felder statt der üblichen Dienstnummer-Felder anzeigen zu lassen. Über diese Felder können Mitarbeiter der eigenen Bezirksstelle direkt per Name oder Dienstnummer bzw. Teilen von diesen gesucht und in den Dienstplan eingetragen werden.

    Diese Funktion ist eine optionale Funktion und muss daher über die Einstellungen der Erweiterung eigens aktiviert werden.

Den kompletten Versionsverlauf können Sie auf unserer Github Projektseite einsehen.

Wir wünschen Ihnen viel Spaß mit der neusten Version unserer Erweiterung!
